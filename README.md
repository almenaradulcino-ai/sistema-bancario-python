# 🏦 Sistema Bancário em Python

Sistema bancário de linha de comando desenvolvido em Python puro, com funcionalidades essenciais de uma conta bancária real.

## ✅ Funcionalidades

- Cadastro de usuário
- Login com autenticação
- Depósito
- Saque
- Consulta de saldo
- Histórico de transações
- Persistência de dados em JSON

## 🛠️ Tecnologias utilizadas

- Python 3
- JSON (armazenamento de dados)
- Módulos próprios separados por responsabilidade

## 📁 Estrutura do projeto

sistema-bancario-python/
├── main.py         # Ponto de entrada e menu principal

├── usuarios.py     # Cadastro e autenticação de usuários

├── operacoes.py    # Lógica de depósito, saque e histórico

└── dados.json      # Banco de dados local em JSON

## ▶️ Como executar

Pré-requisito: Python 3 instalado na máquina.

```bash
# Clone o repositório
git clone https://github.com/almenaradulcino-ai/sistema-bancario-python.git

# Entre na pasta
cd sistema-bancario-python

# Execute
python main.py
```
