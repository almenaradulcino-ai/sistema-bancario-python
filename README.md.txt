# 💰 Sistema Bancário em Python

Projeto desenvolvido para simular operações bancárias reais utilizando Python.

---

## 🚀 Funcionalidades

- Criar conta
- Login de usuário
- Depósito
- Saque
- Ver saldo
- Histórico de transações
- Logout

---

## 🛠️ Tecnologias utilizadas

- Python
- JSON

---

## ▶️ Como executar

1. Baixe o projeto
2. Abra a pasta no terminal
3. Execute:

python main.py

---

## 📚 O que aprendi

- Lógica de programação
- Estrutura de sistemas
- Organização em arquivos
- Manipulação de dados com JSON
